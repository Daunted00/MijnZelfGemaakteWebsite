Kan het gebruikt worden om een website te maken?

De laatste jaren is AI beschikbaar gemaakt voor iedereen en het wordt steeds beter gemaakt. Ik heb AI gebruikt om een pagina voor mijn website te maken om te kijken of de AI het voor elkaar krijgt om een goede website kan maken die geschikt is voor iedereen. De pagina heb ik met ChatGPT gemaakt.

De prompt die ik heb opgegeven: "Ik wil html en css voor een website. Het onderwerp van mijn website is Hoe maak je een website. Als onderwerpen zou ik graag willen dat er uitleg komt over HTML, CSS, UX testing en accessibility. Ik wil graag dat de website responsive is en goede accessibility heeft voor mensen met beperkingen. de style wil ik geinspireerd op de meegeleverde foto van mijn moodboard. Ook wil ik er graag een contactformulier bij hebben." Ook heb ik een moodboard meegegeven om de website naar mijn style te maken.

Reflectie

Als we de wave extensie er weer bij pakken dan zien we dat de website goed scoort. Toch zijn er een paar puntjes die verbeteterd zouden mogen. De website wordt steeds donkerder destemeer je naar beneden gaat. Daardoor kan je het hoofdstuk Contact niet meer goed lezen, terwijl ik wel gevraagd heb om een website die accessible is. Hier heeft AI toch een beetje moeite mee. Ook het contactformulier ziet er niet heel netjes uit. de tekstbalken zijn te lang.

AI werkt vooral goed als je hulpt vraagt voor het genereren of bewerken van code. Ook het responsive maken heeft AI goed gedaan, het staat er dan nog steeds duidelijk. Voor het visueel maken van een website is het nog niet helemaal geschik, maar dat zal in de toekomst wel gaan veranderen. De techniek zal steeds beter worden.
