Interactie

Met Javascript kan je interactie toevoegen aan je website. Dit kan bijvoorbeeld zijn dat je een knop hebt die je kan indrukken en dat er dan iets gebeurd. Dit kan van alles zijn, bijvoorbeeld dat er een plaatje veranderd of dat er een tekst verschijnt. Je kan ook een formulier maken waarbij je een aantal vragen moet beantwoorden en dat er dan een antwoord komt. Er zijn nog veel meer dingen die je kan doen met Javascript, dit zijn maar een paar voorbeelden.

Opdracht

Voor deze opdracht moesten we een website maken waarbij we gebruik maken van Javascript. Ik heb een website gemaakt waarbij je een aantal vragen moet beantwoorden en dat er dan een antwoord komt. Dit is een simpele website, maar het is wel een begin van wat er allemaal mogelijk is met Javascript.
