Wat is HTML?

HTML (HyperText Markup Language) is de basisstructuur van een website. Het definieert de inhoud zoals tekst, afbeeldingen, en links.

Wat is CSS?

CSS (Cascading Style Sheets) bepaalt de stijl van een website, zoals kleuren, lettertypen, en lay-out.

Wat is UX Testing?

UX testing onderzoekt hoe gebruikers interageren met je website om verbeteringen in gebruiksvriendelijkheid aan te brengen.

Wat is Accessibility?

Accessibility zorgt ervoor dat je website toegankelijk is voor mensen met beperkingen, zoals slechtzienden of slechthorenden.
