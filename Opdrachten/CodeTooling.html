Hulpmiddelen

Er zijn verschillende hulpmiddelen die je kunt gebruiken om je website op te maken. Hierbij kun je denken aan css zelfschrijven zoals beschreven in een vorige hoofdstuk, maar we kunnen het ook met css-frameworks doen. Een css-frameworks is een set standaard waarde die gebruikt kunnen worden in de code van de website om het uiterlijk aan te passen. css zet je in een apart bestand, maar css-frameworks kun je meteen toevoegen aan je code van je pagina. Hiermee kan je makkelijker zien wat er voor zorgt dat je pagina er zo uit ziet.

Hoe heb ik het gebruikt?

Ik was al redelijk ver met mijn css. Toen kwam ik er achter dat er een framework in de website moest zitten. Dit is een onderdeel van de toetsbeschrijving. Omdat ik mijn css al redelijk voor elkaar had was het lastiger om een css-framework toe te voegen. Dit schopte de opmaak van mijn site in de war. Daarom heb ik er voor gekozen om op maar 1 pagina een css-framework te gebruiken. Ik heb dat gedaan op deze pagina, de pagina van codetooling. Zo heb ik toch laten zien dat ik het kan gebruiken, maar hoefde er niet helemaal mijn website voor om te gooien
