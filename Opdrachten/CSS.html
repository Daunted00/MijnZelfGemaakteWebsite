CSS

Week 6

Opmaak

Met HTML hebben we een structuur gemaakt van de website, maar daarmee ziet de website er niet goed uit. Daar hebben we CSS voor nodig om de website een style te geven. Met CSS kan er gigantisch veel gedaan worden, bijvoorbeeld:

  • kleuren veranderen
  • lettertype veranderen
  • Aangeven waar een onderdeel moet staan
  • Schaduw geven aan onderdelen
  • plaatje aanpassen naar hoe jij het wilt hebben

Er zijn nog veel meer dingen die er gedaan kunnen worden. Deze voorbeelden zijn nog maar het begin van wat er verder allemaal gedaan kan worden.

In de afbeelding hierboven heb ik geoefend met het CSS voor mijn website. Hierbij heb ik verschillende dingen uitgeprobeerd om het allemaal een beetje onder de knie te krijgen. Dat is denk ik wel aardig gelukt, want mijn huidige website ziet er al een stuk beter uit dan hoe ik begonnen ben.

Voorbeeld

Hier zien we een voorbeeld van een stukje CSS. Dit stukje is de CSS die ik gemaakt heb voor het rondje die te zien is in de titelbalk van mijn website. De hoogte en breedte worden aangegeven voor het juiste formaat. De achtergrond kleur wordt verandert naar licht grijs. De border-radius zorgt ervoor dat het van een vierkant een rondje wordt en de box-shadow geeft het rondje een witte gloed waardoor het lijkt alsof het rondje licht geeft.
