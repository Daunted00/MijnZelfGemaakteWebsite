Wie Ben ik?

Hallo, Mijn naam is Jochem Hogeveen. Ik ben 24 jaar en ik woon in Barneveld. Mijn hobbies zijn gamen, naar de bios gaan en leuke dingen doen met vrienden.

Waarom deze website?

Dit is een website voor een schoolopdracht. Het is een website gericht voor familie en vrienden en het thema is: Hoe maak je een website. Deze website laat de verschillende stappen zien die ik heb genomen om te een website te maken. Deze stappen zijn allemaal te vinden in de opdrachten pagina die te bereiken is via het menu rechtsboven.

Wat voor programma's heb ik gebruikt.

  • Balsamiq: om mijn lo-fi/mid-fi prototype te maken
  • Figma: om mijn hi-fi prototype te maken
  • Visual studio code: om de website zelf in te maken.
  • Zapier: voor het contact-formulier
  • WAVE: om mijn code te controleren op accessibility
  • ChatGPT: voor de pagina gemaakt door ChatGPT

    • Hoe heb ik het aangepakt

    Per les maakte ik mijn content en sloeg het op in word bestanden. Mijn website ben ik eerst begonnen om alles goed te krijgen. Eerst mijn layout goed maken en ook zorgen dat de code goed is waardoor ik alles makkelijk kan hergebruiken.

    Hoe zou ik het de volgende keer anders doen?

    De volgende keer zou ik wel elke week de content die ik gemaakt heb meteen op de website zetten. Hier ben ik laat mee begonnen waardoor het op het einde best veel was. Voor de rest is het allemaal prima verlopen, goeie progressie door de weken heen. Ik ben heel blij met het resultaat. Het ziet er misschien niet heel spectaculair uit, maar ik ben wel trots op mezelf
