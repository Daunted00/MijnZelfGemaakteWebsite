SCSS

SCSS is een uitbreiding van CSS. Het is een manier om CSS te schrijven met variabelen, functies en nog veel meer. SCSS is makkelijker te lezen en te schrijven dan CSS. De SCSS wordt omgezet naar CSS voordat het in de browser komt. Dit gebeurt met een compiler. SCSS is een preprocessor van CSS. Dit betekent dat SCSS code kan bevatten die niet in CSS kan.

Wat heb ik er mee gedaan

Voor mijn website heb ik geen SCSS gebruikt. Mijn CSS was al bijna af voor dat we les kregen in SCSS. Hierdoor heb ik er voor gekozen het zo te laten. Als ik een volgende website zou gaan maken dan ga ik het wel proberen om in SCSS te maken. Daar wordt de CSS leesbaarder van.
