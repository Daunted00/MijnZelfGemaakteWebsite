Search Engine Optimalization

Als je een website hebt dan wil je natuurlijk wel dat mensen het kunnen vinden. Dit gebeurt vaak via zoekmachines zoals google, maar om je website hoog neer te zetten in de lijst zijn er een aantal dingen die je kan doen. Door te zorgen dat je website rustig is en niet te veel informatie bevat, zorg je ervoor dat de website sneller laad en dat de zoekmachine je website hoger neerzet. Ook is het belangrijk dat je website goed werkt op mobiele apparaten, omdat de meeste mensen tegenwoordig via hun telefoon op het internet zitten. Als je website goed werkt op mobiele apparaten, zorgt dit ervoor dat de zoekmachine je website hoger neerzet.

Wat heb ik er voor gedaan

De website kan niet gezien worden door google, dus voor dit project was het niet nodig geweest om de SEO te verbeteren. Toch is het wel handig om er naar te kijken en er zijn een aantal dingen die ik heb gedaan om er voor te zorgen dat de SEO verbeterd is.

    Voor mensen

  • Ik heb een "over mij" pagina gemaakt om mezelf voor te stellen. Dit geeft vertrouwen en daardoor blijven mensen langer op je website. Omdat mensen langer op je website blijven denk google dat jouw website belangrijk is en zet je daardoor hoger neer.
  • De pagina's heb ik geprobeerd eenvoudig te houden. Dit zorgt er voor dat mensen het niet te overweldigend vinden, wat weer kan zorgen voor dat ze snel wegklikken.
  • Ik heb geprobeerd om de tekst interesant te maken. Als mensen in je verhaal komen dan blijven ze langer op je website.
  • De website is ook simpel te navigeren. Als gasten van je website het te moeilijk vinden dan houden ze er mee op en ervaren ze het als onprettig.

    • Voor Machines

  • Door zoveel mogelijk zoekwoorden te gebruiken kan de zoekmachine je op meerdere manieren vinden, daarom heb ik mijn taalgebruik ruim proberen te houden.
  • Als je website toegankelijk is voor mensen die bepaalde dingen slecht of niet kunnen krijg je ook een betere score. Ik heb mijn website geprobeerd heel toegankelijk te maken.
