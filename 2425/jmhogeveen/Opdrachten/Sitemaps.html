Indeling

De sitemap is een overzicht van hoe de pagina's met elkaar verbonden zijn. Hier hebben we visualisatie bijgemaakt en dan kan je nu overzichtelijk zien hoe de website in elkaar steekt. Je begint bij de hoofdpagina en vanaf daar kan je verder in de website bewegen.

Tijdens deze opdracht hebben we ook nagedacht over een aantal vragen die hierbij horen, zoals bijvoorbeeld voor wie deze website is en wat het doel van deze website is.

Het doel van de website is om te laten zien wat er allemaal komt kijken bij het maken van een website. Deze website is gemaakt voor familie zodat zij kunnen begrijpen hoe het werkt om een website te maken.
